  • зұ»WebиҢғејҸзј–зЁӢ

    JS UIжЎҶжһ¶йҮҮз”Ёзұ»HTMLе’ҢCSS Webзј–зЁӢиҜӯиЁҖдҪңдёәйЎөйқўеёғеұҖе’ҢйЎөйқўж ·ејҸзҡ„ејҖеҸ‘иҜӯиЁҖпјҢйЎөйқўдёҡеҠЎйҖ»иҫ‘еҲҷж”ҜжҢҒECMAScriptи§„иҢғзҡ„JavaScriptиҜӯиЁҖгҖӮJS UIжЎҶжһ¶жҸҗдҫӣзҡ„зұ»Webзј–зЁӢиҢғејҸпјҢеҸҜд»Ҙи®©ејҖеҸ‘иҖ…йҒҝе…Қзј–еҶҷUIзҠ¶жҖҒеҲҮжҚўзҡ„д»Јз ҒпјҢи§Ҷеӣҫй…ҚзҪ®дҝЎжҒҜжӣҙеҠ зӣҙи§ӮгҖӮ

еӣҫ 1 JS UIжЎҶжһ¶жһ¶жһ„

JS UIжЎҶжһ¶еҢ…жӢ¬еә”з”ЁеұӮпјҲApplicationпјүгҖҒеүҚз«ҜжЎҶжһ¶еұӮпјҲFrameworkпјүгҖҒеј•ж“ҺеұӮпјҲEngineпјүе’Ңе№іеҸ°йҖӮй…ҚеұӮпјҲPorting LayerпјүгҖӮ

  • Application

    еә”з”ЁеұӮиЎЁзӨәејҖеҸ‘иҖ…дҪҝз”ЁJS UIжЎҶжһ¶ејҖеҸ‘зҡ„FAеә”з”ЁпјҢиҝҷйҮҢзҡ„FAеә”з”Ёзү№жҢҮJS FAеә”з”ЁгҖӮ

  • Framework

    еүҚз«ҜжЎҶжһ¶еұӮдё»иҰҒе®ҢжҲҗеүҚз«ҜйЎөйқўи§ЈжһҗпјҢд»ҘеҸҠжҸҗдҫӣMVVMпјҲModel-View-ViewModelпјүејҖеҸ‘жЁЎејҸгҖҒйЎөйқўи·Ҝз”ұжңәеҲ¶е’ҢиҮӘе®ҡд№үз»„д»¶зӯүиғҪеҠӣгҖӮ

  • Engine

    еј•ж“ҺеұӮдё»иҰҒжҸҗдҫӣеҠЁз”»и§ЈжһҗгҖҒDOMпјҲDocument Object Modelпјүж ‘жһ„е»әгҖҒеёғеұҖи®Ўз®—гҖҒжёІжҹ“е‘Ҫд»Өжһ„е»әдёҺз»ҳеҲ¶гҖҒдәӢд»¶з®ЎзҗҶзӯүиғҪеҠӣгҖӮ

  • Porting Layer

    йҖӮй…ҚеұӮдё»иҰҒе®ҢжҲҗеҜ№е№іеҸ°еұӮиҝӣиЎҢжҠҪиұЎпјҢжҸҗдҫӣжҠҪиұЎжҺҘеҸЈпјҢеҸҜд»ҘеҜ№жҺҘеҲ°зі»з»ҹе№іеҸ°гҖӮжҜ”еҰӮпјҡдәӢд»¶еҜ№жҺҘгҖҒжёІжҹ“з®ЎзәҝеҜ№жҺҘе’Ңзі»з»ҹз”ҹе‘Ҫе‘ЁжңҹеҜ№жҺҘзӯүгҖӮ
