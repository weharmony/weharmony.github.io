返回值：

类型

说明

音频管理类。

示例：

var audioManager = audio.getAudioManager();

AudioVolumeType

音频流类型。

名称

默认值

描述

MEDIA

1

表示媒体。

RINGTONE

2

表示铃声。

DeviceFlag

可获取的设备种类。

名称

默认值

描述

OUTPUT_DEVICES_FLAG

1

表示输出设备种类。

INPUT_DEVICES_FLAG

2

表示输入设备种类。

ALL_DEVICES_FLAG

3

表示所有设备种类。

DeviceRole

设备角色。

名称

默认值

描述

INPUT_DEVICE

1

OUTPUT_DEVICE

2

设备类型。

名称

默认值

描述

INVALID

0

无效设备。

SPEAKER

1

扬声器。

WIRED_HEADSET

2

有线耳机。

BLUETOOTH_SCO

3

SCO（Synchronous Connection Oriented Link）蓝牙设备。

BLUETOOTH_A2DP

4

A2DP（Advanced Audio Distribution Profile ）蓝牙设备。

MIC

5

麦克风。

setVolume(AudioVolumeType, number, AsyncCallback<void>)

设置某个流的音量，使用callback方式作为异步方法。

参数：

参数名

类型

必填

说明

audioType

AudioVolumeType

是

流类型。

volume

number

是

音量。

callback

AsyncCallback<void>

是

回调表示成功还是失败。

示例：

audioManager.setVolume(audio.AudioVolumeType.MEDIA, 30, (err)=>{
   if (err) {
	   console.error(`failed to set volume ${err.message}`);
	   return;
   }
   console.log(`Media setVolume successful callback`);
})
