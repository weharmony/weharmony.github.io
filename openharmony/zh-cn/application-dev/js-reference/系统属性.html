获取系统属性Key对应的值。

参数：

参数名

类型

必填

说明

key

string

是

待查询的系统属性Key。

def

string

否

默认值。

返回值：

类型

说明

string

系统属性值，若key不存在，返回默认值。若未指定默认值，返回空字符串。
